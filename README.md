# react + electron 人工智能图像识别前端开发环境搭建

## 技术栈

react + react-router + axios + antd + es6 + OpenSeadragon

## 项目运行
安装依赖:

```shell
$ npm install
```

启动web版本开发环境

```shell
$ npm start
```

运行electron开发环境

```shell
$ npm run dev
```

构建

```shell
$ npm run build
```

打包windows 64bit

```shell
$ npm run package
```
